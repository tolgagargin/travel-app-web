# Trip Planner — web demo

**Try it: https://tolgagargin.github.io/travel-app-web/**

This repo only hosts the compiled Flutter web build of a private travel-planning
app (flights, hotels, day-by-day itineraries, budgets and a social layer, in
Turkish and English). The demo runs on built-in sample data — no sign-up needed.

The `gh-pages` branch is overwritten automatically by the source repo's CI on
every release; nothing here is edited by hand.
